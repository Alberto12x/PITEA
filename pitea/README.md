# TFG-PITEA

## UV 
Uv es un gestor de paquetes de python moderno

[Documentacion uv](https://github.com/astral-sh/uv)

Basico :

Crear entorno 

```uv init```

Añadir biblotecas 

```uv add```

Ejecutar 

```uv run nombre_archivo.py```
